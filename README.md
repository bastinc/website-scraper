# Website-scraper

A tool to scrap webpages from a domain root url

## Requirements

    * Python >= 3.8
    * [Poetry](https://python-poetry.org/)

## Setup

Create a new virtualenv

```bash
poetry shell
```

```bash
poetry install
```

## Run

```bash
poetry run website-scraper <URL>
```

## Build

```bash
poetry build
```
